markdown图片

网上的图片链接可以直接使用，下面对本地图片使用进行配置。

使用

安装插件npm install https://github.com/CodeFalling/hexo-asset-image --save

在_config.yml （不是next里面的_config.yml）中设置permalink: :year/:month/:day/:title/，设置 post_asset_folder: true，这主要将文件拷贝到/public文件夹。其中permalink很关键，只能这样设置。
