2022/12/11

前言

当今时代，绝大多数企业的应用都是运行在 Linux 操作系统上，所以对应用进行性能诊断和性能优化时，离不开 Linux 的各种性能观测工具和性能优化工具。

笔者使用过的常见的 Linux 性能观测和性能优化工具有：

  • top/uptime
  • ps/pstree
  • df/du/free/lsblk
  • ip/ifconfig/ping/telnet
  • route/dig/nslookup
  • lsof/netstat/ss
  • tcpdump/tshark/wireshark
  • netstat/vmstat/iostat/pidstat/dstat/mpstat
  • sar/sysctl/ethtool
